# Student Information System (Dockerized)

This project is developed for the System Programming course.
The goal is to build a student information system using C++, PostgreSQL, and Docker.

## System Architecture
- PostgreSQL database container
- C++ application container
- Docker Compose for orchestration

## Team Roles
- Project Manager
- Docker & DevOps Engineer
- C++ Developer
